# openapi-usage ドキュメント

このディレクトリにはパッケージ公開者向けのドキュメントが含まれています。

## 目次

1. [PUBLISHING.md](./PUBLISHING.md) - npm公開の初期設定手順
2. [RELEASING.md](./RELEASING.md) - リリース手順（日常的な作業）
3. [SECRETS.md](./SECRETS.md) - GitHub Secretsの設定方法

## クイックスタート

初めてこのリポジトリでリリースを行う場合は、以下の順序で設定を行ってください:

1. [SECRETS.md](./SECRETS.md) を参照してシークレットを設定
2. [PUBLISHING.md](./PUBLISHING.md) を参照して初期設定を完了
3. [RELEASING.md](./RELEASING.md) を参照してリリースワークフローを理解
